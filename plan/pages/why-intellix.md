# Why Intellix

**Purpose:** Differentiation vs generic “computer classes” and vs other local offerings — **more practical + job-focused**.

## Content pillars

| Pillar | Message |
| :--- | :--- |
| **Project-based curriculum** | Real assignments and a **mandatory portfolio** — not slides-only |
| **Career & income paths** | Freelancing (Upwork, Fiverr), employable skills, optional **job placement / internship** language as programs mature |
| **Modern tools** | Industry-standard apps: **Photoshop**, **Illustrator**, **InDesign**, **Canva**; **Premiere Pro**, **Final Cut Pro** (Mac), **DaVinci Resolve**, **CapCut**; **Lightroom** for photo; **DJI Osmo Pocket 3** + **DJI Mimo** and pro audio on set; **Cursor** / **VS Code**, **Git**, **Docker**, **Xcode** / **Android Studio** by path; **ChatGPT** / **Claude** and workflow automation |
| **Human scale** | Small cohorts where possible; instructor access |
| **Reliable learning environment** | Strong internet, **backup power**, comfortable classroom — signals you respect students’ time (especially relevant in The Gambia) |

## Visuals

Simple infographic “course → outcome”; mini student stories; optional “before / after” skill snapshots.
