# Blog / resources (optional, high leverage)

**Purpose:** SEO, authority, and nurture — especially for **WhatsApp-shared** links.

**IA alignment:** Treat this hub as the **Resources** column in [`site-navigation.md`](site-navigation.md) — articles, cheatsheets, project write-ups, and (later) video links.

## Resource types (footer-style labels)

| Type | What to publish | Example |
| :--- | :--- | :--- |
| **Articles** | Longer guides, stories | “First month as a freelance editor in Banjul” |
| **Blog** | Shorter updates, cohort news | New **EditPro** cohort dates |
| **Cheatsheets** | PDF / one-pagers | Poster sizes, shortcut keys for Premiere, SQL basics |
| **Projects** | Student case studies | Deep links to [Portfolio](portfolio.md) entries with “how I built this” |
| **Videos** | Embedded or YouTube | Tool tips, guest talks |
| **Docs / FAQs** | Static help | Payment plans, BYOD camera policy — can live here or on Contact |

## Topic clusters

- Design tips for beginners; **poster vs social** sizing cheat sheets
- Video editing workflow; **commercial vs creator** pacing; gear-on-a-budget
- **Photography:** light on a budget, event vs portrait kit
- **Videography:** audio on set, shot lists for small crews
- First **Fiverr / Upwork** gig
- **AI tools** for productivity (2026 framing)
- Personal brand and social media for freelancers
- “Computer basics” series feeding Essential Skills course
- Software career paths: frontend vs full stack, first deploy, mobile vs iOS, intro to **DevOps** / CI

## Format

Cards with featured image, clear title, internal links to [Courses](courses.md) and [Enroll](enroll.md).
