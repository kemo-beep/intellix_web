# Pixel Mastery — graphic design

**Program:** Pixel Mastery · **Parent:** [Courses overview](courses.md)

One **Pixel Mastery** umbrella; learners pick a **track** (or stack tracks over time). Each track ends with **portfolio-ready deliverables** (print PDFs, export packs, mockups).

## Tracks

| Track | Focus | Example deliverables | **Tools you’ll learn & use** |
| :--- | :--- | :--- | :--- |
| **Advertising & campaigns** | Banners, display, promo layouts, A/B variants | Ad sets for web/print, simple campaign board | **Adobe Photoshop**, **Illustrator**; **Canva** for fast variants and resize; optional **Figma** for collaboration |
| **Social media design** | Feeds, stories, carousels, thumbnails, platform specs | Full month-style content pack for one brand | **Canva**, **Photoshop**; templates and safe zones for Meta / TikTok / YouTube |
| **Posters & large format** | Posters, flyers, event boards, typography hierarchy | Print-ready poster + digital versions | **Photoshop**, **Illustrator**, **InDesign** (multi-page flyers); export for print PDF |
| **Brand & identity** | Logos, color/type systems, basic brand guidelines | Mini brand kit for a fictional or student brand | **Illustrator** (vector logos), **Photoshop** (mockups), **InDesign** or **PDF** for one-pager guidelines |
| **Editorial & layout** | Magazines, reports, multi-page documents | Short editorial spread or report template | **InDesign**, **Illustrator** (charts/icons), **Photoshop** (image prep) |

## Program core (all Pixel tracks)

**Adobe Creative Cloud** focus where licensed — **Photoshop**, **Illustrator**, **InDesign** as needed; **Canva** for speed and client handoff; optional **Figma** if you teach UI-adjacent layouts.

## Site UX

Tabs or accordions per track; shared **Design foundations** module optional for absolute beginners.
